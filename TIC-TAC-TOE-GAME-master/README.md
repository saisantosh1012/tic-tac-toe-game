## TIC TAC TOE GAME BY D SAI SANTOSH
 - `ReactJS`
 - `Framer Motion`



```
# Clone this repository
$ git clone https://github.com/YoucefHmd/TIC-TAC-TOE-GAME.git

# Go into the repository
$ cd TIC-TAC-TOE-GAME

# Install dependencies
$ npm install

# Run the app
$ npm start
```
